# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.4.x   | Yes       |
| 1.3.x   | No        |
| < 1.3   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in `alterion-encrypt`, **please do not open a public GitHub issue.**

Report it privately by emailing:

**security@alterion.dpdns.org**

Please include:
- A description of the vulnerability
- Steps to reproduce or a proof-of-concept
- The version(s) affected
- Any potential impact or severity assessment

You can expect an acknowledgement within **72 hours**. We will work with you to understand and resolve the issue, and will coordinate a disclosure timeline.
